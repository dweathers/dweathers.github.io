(a) What is the probability that the password contains all letters?

Passwords are typically case sensitive, so our alphabet consists of 62 characters and a password with all letters will be made from 52 characters. (It was not 100% clear in the problem, so if you replace these numbers with 36 and 26 respectively, that is okay.)

By counting the passwords and using the definition of probability, then

P(all letters)=526626=0.348.P(\text{all letters}) = \dfrac{52^6}{62^6} = 0.348.P(all letters)=626526​=0.348.

(b) What is the probability that the password has no repeated characters?

The number of passwords with no replacement is P(62,6)=62!/56!P(62,6)=62!/56!P(62,6)=62!/56! (PPP here referring to a permutation, not a probability). Therefore

P(no repetition)=62!/56!626=0.779P(\text{no repetition}) = \dfrac{62!/56!}{62^6} = 0.779P(no repetition)=62662!/56!​=0.779

Our sample space is now just the set of passwords with no repetition; recall there are 62!/56!62!/56!62!/56! such passwords. If we consider only the ones with only letters, there are 52!/46!52!/46!52!/46! such passwords. Therefore
