15.6 Can a relation be both reflexive and antireflexive? If so, give an example; if not, give an explanation.

Suppose that the underlying set contains the element aaa. Then the relation, being reflexive, contains (a,a)(a,a)(a,a); being antireflexive it must not contain (a,a)(a,a)(a,a). This obvious contradiction leads us to conclude that no such relation exists on a nonempty set.

However… we do have an empty set! So the empty relation R=∅R=\varnothingR=∅ over the empty set ∅\varnothing∅ is not only an example but it is the only example.

If this relation is confusing to you, it is the relation where nothing is related because there is nothing to relate. This is the only relation on the empty set. (The product of the empty set and itself is empty, and so its only subset is the empty set.) It is both reflexive because everything in the set is related to itself and it is antireflexive because every element in the underlying set is not related to itself. (See Chapter 1 for a discussion of such “vacuous truths”.)

The empty relation on the empty set is not the same as the empty relation over a nonempty set, because that relation would not be reflexive (nothing from the nonempty underlying set is related to itself.) If this relation is still confusing…well, that’s okay, it’s a confusing relation.

16.5 What is the reflexive closure of the relation described by x<yx < yx<y on R\mathbf{R}R?

One way to approach this problem is to use the list of pairs from 15.1(e), close that set under reflexiveness, and infer the appropriate operation from the resulting list of pairs.

We can also proceed conceptually. The reflexive closure of any relation is said relation’s union with the diagonal relation. The diagonal relation is exactly the set of pairs (x,y)(x,y)(x,y) that satisfy the property x=yx=yx=y.

Recalling that a union is an “or” statement, we are looking for the set of pairs where x<yx<yx<y or x=yx=yx=y. This is exactly the relation of pairs where x≤yx \le yx≤y.
