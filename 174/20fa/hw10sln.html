14.4 Consider a family of sets that together comprise the power set of some known set XXX. Describe how this family forms a Boolean algebra: Identify the join, meet, and complement operations, which object would play the role of 111, and which object would play the role of 000.

Intersection is meet, union is join, and (surprise!) complement is complement. Why? Well, if you have already made the connection between (say) meet and conjunction from logic, then it is a short hop from conjunction to intersection. In other words, a∧b=1a \wedge b = 1a∧b=1 if and only if both aaa and bbb are 1. An element xxx is in E∩FE \cap FE∩F if and only if xxx is in both EEE and FFF.

A good naive guess at the element that plays the role of 000 is ∅\varnothing∅, and that guess is correct: ∅∪E=E\varnothing \cup E = E∅∪E=E for any set EEE (just like 0∧a=00 \wedge a = 00∧a=0 for all elements aaa of a Boolean algebra). If all of these sets are contained in XXX, then ∅‾=X\overline{\varnothing} = X∅​=X. The fact that 000 and 111 are complements encourages us to suggest XXX as the 111-type object, and sure enough X∩E=EX \cap E = EX∩E=E for any set EEE (assuming EEE was a subset of XXX to begin with). These five identifications are how a power set becomes a Boolean algebra. (In Chapter 18, you can prove that any finite lattice is a Boolean algebra, and that will also get you there.)
