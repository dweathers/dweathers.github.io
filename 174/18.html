In the last chapter we looked at equivalence relations, which generalize the idea of equality. Partial orders will do the same for inequalities. How can we say one object is “less than” another when the two objects are not numbers? You may be able to think of a way to order sets, or strings, or matrices. That way is probably a partial order.

18.1 Introduction

The secret ingredient to order is that an order must only go one way. If xxx is less than yyy, the only way that yyy should be able to be less than xxx is if the two are equal (and by “less than” we really mean “less than or equal to”). Fortunately, we have a word for this: antisymmetry!

Definition 18.1.1 A relation is a partial order if it is reflexive, antisymmetric, and transitive. If RRR is a partial order on a set XXX and (x,y)∈R(x,y) \in R(x,y)∈R, we write x≤Ryx \le_R yx≤R​y, x⪯yx \preceq yx⪯y, or x≤yx \le yx≤y. The third is only used when the partial order is clear from context.

Examples. The prototypical partial order is the relation x≤yx \le yx≤y on the integers. This relation is reflexive (any integer is less than or equal to itself), antisymmetric (x≤yx \le yx≤y and y≤xy \le xy≤x means x=yx=yx=y), and transitive (as we saw in Chapter 8).

Given a family of sets, there are two “natural” ways to order them: by inclusion, or by size. The first is the relation where sets EEE and FFF are related if and only if E⊆FE \subseteq FE⊆F. The other is given by ∣E∣≤∣F∣|E| \le |F|∣E∣≤∣F∣. In Chapter 15 you verified that the former was a partial order; you will verify the latter in this chapter.

Given a set of integers we may say that mmm is related to nnn if and only if m∣nm|nm∣n. In Chapter 4 you proved that this is a partial order (even if you didn’t know it at the time).

Why the word “partial”? Take the last example, the order defined by divisibility, and suppose our underlying set is all of Z\mathbf{Z}Z. Neither 3∣53|53∣5 nor 5∣35|35∣3. Since we have two elements that cannot be ordered, the order is only partial. The following definitions formalize the notion.

Definition 18.1.2 Suppose that RRR is a partial order on a set XXX. If x⪯yx \preceq yx⪯y or y⪯xy \preceq xy⪯x in the partial order, we say xxx and yyy are comparable; if not, they are incomparable.

Definition 18.1.3 An order on a set XXX is total or linear if xxx and yyy are comparable for every x,y∈Xx,y \in Xx,y∈X.

Example. The usual order on the integers is a total order. The divisibility order is total if restricted to the right set, for example, the set of powers of 222.

Definition 18.1.4 A set XXX is well-ordered if it has a total order ⪯\preceq⪯ and an element x0x_0x0​ such that x0⪯xx_0 \preceq xx0​⪯x for all x∈Xx \in Xx∈X.

In language we will learn later, a well-ordered set is a set with a total order and a “minimum” element.

Example. The poster child for well-ordered sets is N\mathbf{N}N, with the usual total order and the least element 000. Observe that one of the most important features of N\mathbf{N}N, the validity of mathematical induction, is not special to N\mathbf{N}N at all: all we need is somewhere to start and a way to move “forward” through the set. So, well-ordered sets are precisely those for which mathematical induction is possible.

This visual is, in a word, awful. There are too many arrows! However, the fact that we are dealing with a partial order implies that certain arrows are redundant. Since every partial order is reflexive, we can lose the loops. Definition 18.2.2 The Hasse diagram of a partial order is a graph on a set XXX where there is an edge from a vertex xxx up to yyy if and only if yyy covers xxx in the order.

18.3 Extreme features of a partial order

We will close by discussing various “extreme” features a partial order can have.

Definition 18.3.1 Let XXX be a partially ordered set. We say that xxx is less than yyy (in the partial order) if x⪯yx \preceq yx⪯y, and then yyy is greater than xxx.

This definition might seem weird, but remember that we are not necessarily talking about numbers. Once everyone knows which partial order is being discussed, we use “less than” and “greater than” to describe relations within the order.

Definition 18.3.2 An element xxx in a partially ordered set XXX is minimal/maximal if it is not greater/less than any other element. If the minimal/maximal element is unique, it is called the minimum/maximum.

Example. It is important to note that minimal does not mean less than everything; it means not greater than anything, and these are not the same! Above, www is minimal, but it is not less than zzz. (Likewise zzz is maximal even though it is not greater than www.)

Definition 18.3.3 Fix a subset AAA of the partially ordered set XXX. A lower bound/upper bound of AAA is any element of XXX that is less/greater than every element in AAA.

Example. Consider the order {(a,a),(a,b),(a,c),\{(a,a), (a,b), (a,c),{(a,a),(a,b),(a,c), (a,d),(a,e),(a,f),(b,b),(a,d), (a,e), (a,f), (b,b),(a,d),(a,e),(a,f),(b,b), (b,c),(b,d),(b,e),(b,f),(b,c), (b,d), (b,e), (b,f),(b,c),(b,d),(b,e),(b,f), (c,c),(c,e),(c,f),(d,d),(c,c), (c,e), (c,f), (d,d),(c,c),(c,e),(c,f),(d,d), (d,e),(d,f),(e,e),(f,f),(d,e), (d,f), (e,e), (f,f),(d,e),(d,f),(e,e),(f,f), (g,d),(g,e),(g,f),(g,g)}(g,d), (g,e), (g,f), (g,g)\}(g,d),(g,e),(g,f),(g,g)} and the subset A={b,c,d}A=\{b,c,d\}A={b,c,d} of the ordered set.

LOWER BOUND EXAMPLE!

The upper bounds of AAA are the elements eee and fff, because both are greater than every element in AAA.

The lower bounds of AAA are aaa and bbb. Since partial orders are reflexive, bbb is “less than” (remember, really “less than or equal to”) itself. Because ggg is only less than ddd, it is not a lower bound of AAA.

Definition 18.3.4 Let AAA be a subset of a partially ordered set. The greatest lower bound of AAA, if it exists, is the maximum element of the set of lower bounds of AAA. The greatest lower bound is denoted glb\text{glb}glb and sometimes called the infemum, or inf\text{inf}inf, of the set. (The least upper bound, lub\text{lub}lub, supremum, or sup\text{sup}sup is the minimum upper bound.)

Sound familiar? Indeed, we have talked about finding “tightest bounds” when we discussed growth rates of sequences all the way back in Chapter 7.

You may have noticed that some Hasse diagrams tend to “sprawl,” while others have pleasing geometric shapes. Our final definition puts some structure on that idea.

Definition 18.3.5 A partially ordered set is a lattice if any pair of elements has both a least upper bound and a greatest lower bound.

Examples. The set {a,b,c,d,e,f,g}\{a,b,c,d,e,f,g\}{a,b,c,d,e,f,g} with the order {(a,a),(a,b),(a,c),\{(a,a), (a,b), (a,c),{(a,a),(a,b),(a,c), (a,d),(a,e),(a,f),(b,b),(a,d), (a,e), (a,f), (b,b),(a,d),(a,e),(a,f),(b,b), (b,c),(b,e),(b,f),(b,c), (b,e), (b,f),(b,c),(b,e),(b,f), (c,c),(c,e),(c,f),(d,d),(c,c), (c,e), (c,f), (d,d),(c,c),(c,e),(c,f),(d,d), (d,e),(d,f),(e,e),(f,f),(d,e), (d,f), (e,e), (f,f),(d,e),(d,f),(e,e),(f,f), (g,d),(g,e),(g,f),(g,g)}(g,d), (g,e), (g,f), (g,g)\}(g,d),(g,e),(g,f),(g,g)} from the last two examples is not a lattice, because (for instance) the pair {e,f}\{e,f\}{e,f} fails to have a least upper bound.

An example of a lattice is the inclusion order on any power set; see Section 18.2 for an example. Surprisingly (or maybe not), any linear order is a lattice, even though they don’t visually “look like” lattices. Take a pair {a,b}\{a,b\}{a,b} in a linear order where (say) a⪯ba \preceq ba⪯b (it’s either that or the other way around). Then, the greatest lower bound of the pair is aaa and the least upper bound of the pair is bbb.

Any Boolean algebra yields a lattice, and any finite lattice yields a Boolean algebra. This is an exercise.

Takeaways

  • A partial order generalizes the idea of inequality to objects that aren’t numbers.
  • Examples include inclusion of sets and divisibility of integers.
  • A partial order can be visually represented by a Hasse diagram.
  • There are several ways to discuss extreme elements in a partial order: minimal/maximal elements, minimum/maximum elements, and lower/upper bounds.
