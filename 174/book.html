0. What is discrete mathematics?

0.1 It’s math you do quietly, right?

The phrase “discrete mathematics” is often used nebulously to refer to a certain branch of mathematics that, from the college point of view, is “not calculus” or perhaps is applicable to computer science. In this section, we will try to understand what it means for a topic to fall under the umbrella of discrete mathematics.

0.2 A tale of two sets

Throughout high school and college mathematics so far, you have largely been working with R\mathbf{R}R, the set of real numbers. This set includes virtually every number you can imagine, including 000, −1-1−1, 3/43/43/4, 0.60.60.6, 2\sqrt{2}2​, and π\piπ. (If you know what a complex number is, R\mathbf{R}R does not include those.)

One of the distinguishing features of R\mathbf{R}R is their density. Here’s the idea. If I give you any two real numbers – say, 000 and 111 – you can find one in between – say, 1/π1/\pi1/π. Then between 000 and 1/π1/\pi1/π you can find another. And so on.

Another set with which you are familiar is N\mathbf{N}N, the set of natural numbers. The natural numbers are the counting numbers starting with 000 (in this book, anyway). If I give you two natural numbers like 000 and 111, you cannot find another natural in between. In this sense, the natural numbers are – you guessed it! – discrete. We often call N\mathbf{N}N countable and R\mathbf{R}R uncountable, because the elements of the latter cannot be listed. (You will never get to 1.)

So, discrete mathematics is the study of objects that are countable.

0.3 Some examples

Here are some questions we might try to answer using discrete mathematics. In fact, you will be able to answer all these questions by the end of the book.

Suppose we have a program that will activate condition 4 if conditions 1 and 2 are met but not condition 3. How do we know when condition 4 is active?

We can use logic to represent this program with the statement (p∧q∧¬r)→s(p \wedge q \wedge \neg r) \to s(p∧q∧¬r)→s, and then use truth tables to determine exactly when condition 4 will trigger.

Which of these two programs is more efficient?

A sequence is a function of natural numbers. An example is the number (an)(a_n)(an​) of steps it takes to run a computer program with nnn inputs. If (an)(a_n)(an​) grows must faster than (bn)(b_n)(bn​) does, that can tell us that one program scales worse for large quantities of data.

Some members of a community are sick. How do we know who is at risk of being infected?

Relations encode relationships between sets of objects. We may compose relations to iterate those relationships over and over again. If person xxx is sick and has come into contact with person yyy, and yyy has come into contact with person zzz, and zzz has since seen www, then www is at risk.

1. Sets

1.1 Membership

For nearly all interested parties, sets form the foundation of all mathematics. Every mathematical objects you have ever encountered can be described as a set, from numbers themselves to functions and matrices.

We will say that a set is an unordered collection of objects, called its elements. When xxx is an element of the set XXX, we write x∈Xx \in Xx∈X.

Examples. Our starting point will be the set of all natural numbers,

N={0,1,2,3,…}. \mathbf{N} = \{0,1,2,3,\ldots \}.N={0,1,2,3,…}.

Certain sets are special enough to get their own names, with bold or “blackboard bold” face. Sometimes (especially in ink) you will see N\mathbb{N}N instead. Note that the elements of the set are enclosed in curly braces. Proper notation is not optional; using the wrong notation is like saying “milk went store” to mean “I went to the store to get milk.” Finally, not everyone agrees that 000 is a natural number, but computer scientists do.

If we take N\mathbf{N}N and include its negatives, we get the integers

Z={…,−2,−1,0,1,2,…}. \mathbf{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots \}.Z={…,−2,−1,0,1,2,…}.

Notice that in the case of an infinite set, or even a large finite set, we can use ellipses when a pattern is established. One infinite set that defies a pattern is the set of prime numbers. You may recall that a prime number is a positive integer ppp with exactly two factors: itself and 111. We may list a few: 2,3,5,7,11,13,17,19,23,2, 3, 5, 7, 11, 13, 17, 19, 23,2,3,5,7,11,13,17,19,23, but there is no pattern that allows us to employ ellipses.

Fortunately there is another way to write down a set. All previous examples have used roster notation, where the elements are simply listed. It may be more convenient to use set-builder notation, where the set is instead described with a rule that exactly characterizes this element. In that case, the set of primes can be written as

{ p ∣  p is a prime number}. \{ \, p \, | \; p \text{ is a prime number} \}.{p∣p is a prime number}.

To provide one last example, consider the set of all U.S. states. In roster notation this set is a nightmare: {AL, AK, AZ, AR, CA, CO, CT, DE, FL, GA, HI, ID, IL, IN, IA, KS, KY, LA, ME, MD, MA, MI, MN, MS, MO, MT, NE, NV, NH, NJ, NM, NY, NC, ND, OH, OK, OR, PA, RI, SC, SD, TN, TX, UT, VT, VA, WA, WV, WI, WY}. But in set-builder notation we simply write

{ S ∣ S is a U.S. state}. \{ \, S \, | \, S \text{ is a U.S. state} \}.{S∣S is a U.S. state}.

Definition 1.1.1. Two sets XXX and YYY are equal if they contain exactly the same elements, in which case we write X=YX=YX=Y.

Remember that sets are unordered, so even {a,b}={b,a}\{a,b\}=\{b,a\}{a,b}={b,a}.

Definition 1.1.2. Let XXX be a finite set. Its cardinality, denoted ∣X∣|X|∣X∣, is the number of elements in XXX.

Example. If X={a,b,c}X=\{a,b,c\}X={a,b,c}, then ∣X∣=3|X|=3∣X∣=3.

We can talk about the cardinality of infinite sets, but that is far beyond the scope of this course.

1.2 Containment

One of the most important ideas when dealing with sets is that of a subset.

Definition 1.2.1. Let XXX and YYY be sets. We say XXX is a subset of YYY if every element of XXX is also an element of YYY.

Examples. Put X={a,b,c}X = \{a,b,c\}X={a,b,c}, Y={a,c}Y = \{a,c\}Y={a,c}, and Z={b}Z=\{b\}Z={b}. Then Y⊆XY \subseteq XY⊆X, Z⊆XZ \subseteq XZ⊆X, but Y⊈ZY \not\subseteq ZY​⊆Z and Z⊈YZ \not\subseteq YZ​⊆Y.

In mathematics, true statements are either definitions (those taken for granted) or theorems (those proven from the definitions). We will learn to prove more difficult theorems later, but for now we can prove a simple one.

Theorem 1.2.2. Every set is a subset of itself.

A proof is simply an argument that convinces its reader of the truth of a mathematical fact. Valid proofs must only use definitions and the hypotheses of the theorem; it is not correct to assume more than you are given. So the ingredients in the proof of this theorem are simply going to be a set and the definition of the word “subset.”

Proof. Let XXX be a set. Suppose xxx is an arbitrary element of XXX. Then, xxx is also an element of XXX. Because XXX was shown to have any arbitrary element of XXX, then X⊆XX \subseteq XX⊆X. □\square□

Notice that the actual make-up of the set XXX is irrelevant. If XXX has an element, then XXX also has that element, and that is all it takes to conclude X⊆XX \subseteq XX⊆X. It would be incorrect to try to prove X⊆XX \subseteq XX⊆X from an example: just because {0}\{0\}{0} is a subset of itself doesn’t necessarily mean that {0,1}\{0,1\}{0,1} is!

We will discuss logic and proof-writing at length as we continue. For now, let’s continue learning about sets and proving the “simple” theorems as we go.

Definition 1.2.3. The set with no elements is called the empty set, denoted {}\{ \}{} or ∅\varnothing∅.

With a little logical sleight-of-hand, we can prove another (much more surprising) theorem.

Theorem 1.2.4. The empty set is a subset of every set.

Proof. Let XXX be a set. We may conclude ∅⊆X\varnothing \subseteq X∅⊆X if every element of ∅\varnothing∅ is an element of XXX. Observe that this statement is equivalent to saying that no elements of ∅\varnothing∅ are not elements of XXX. Because there are no elements of ∅\varnothing∅ at all, this statement is true; and we are done. □\square□

Perhaps you already agree with the central claim of the proof, that there are no difference between the statements "everything in ∅\varnothing∅ is in XXX" and "nothing in ∅\varnothing∅ is not in XXX". If you do not, take it on faith for now. When we study predicate logic later you will have the tools to convince yourself of the claim.

Such a statement that relies on the emptiness of ∅\varnothing∅ is said to be vacuously true.

When X⊆YX \subseteq YX⊆Y, we say YYY contains XXX or that YYY is a superset of XXX. With the idea of containment in hand we can construct a new type of set.

Definition 1.2.5. Let XXX be a set. Its power set is the set P(X)\mathcal{P}(X)P(X) whose elements are the subsets of XXX. That is,
P(X)={ E ∣ E⊆X }. \mathcal{P}(X) = \{ \, E \, | \, E \subseteq X \, \}.P(X)={E∣E⊆X}.

Wait - our elements are sets? Yes, and this is perfectly fine! Elements can be all sorts of things, other sets included. When we have a “set of sets” we typically refer to it as a family or a collection to avoid using the same word and over and over.

Example. Let X={a,b,c}X = \{a,b,c\}X={a,b,c}. Then

P(X)={∅,{a},{b},{c},{a,b},{a,c},{b,c},X}. \mathcal{P}(X) = \{ \varnothing, \{a\}, \{b\}, \{c\}, \{a,b\}, \{a,c\}, \{b,c\}, X\}. P(X)={∅,{a},{b},{c},{a,b},{a,c},{b,c},X}.

Notice that the number of sets in P(X)\mathcal{P}(X)P(X) can be derived from the number of elements in XXX.

Theorem 1.2.6. If XXX is finite, then ∣P(X)∣=2∣X∣|\mathcal{P}(X)|=2^{|X|}∣P(X)∣=2∣X∣.

We will come back to prove this theorem much later, once we have learned how to count!

1.3 The algebra of sets

You are used to word “algebra” as it refers to a class where you must solve some equations for xxx. If you think about those classes, they are really about understanding operations and functions of real numbers. Think about it: a successful algebra student will know how to manipulate sums, products, polynomials, roots, and exponential functions – ways of combining and mapping real numbers.

So, “algebra” in general refers to the ways objects may be combined and mapped. In this section, we will see many ways that sets can be combined to produce new sets.

Definition 1.3.1. Let XXX and YYY be sets. Their union is the set X∪YX \cup YX∪Y of all elements that are in at least one of XXX or YYY; in set builder notation this is
X∪Y={ x ∣ x∈X or x∈Y}. X \cup Y = \{\, x \, | \, x \in X \text{ or } x \in Y \}.X∪Y={x∣x∈X or x∈Y}.

Definition 1.3.2. Let XXX and YYY be sets. Their intersection is the set X∪YX \cup YX∪Y of all elements that are in both XXX and YYY; in set builder notation this is
X∩Y={ x ∣ x∈X and x∈Y}. X \cap Y = \{\, x \, | \, x \in X \text{ and } x \in Y \}.X∩Y={x∣x∈X and x∈Y}.

A useful way to visualize combinations of sets is with a Venn diagram. The Venn diagrams for union and intersection are shown below.

Example. Let X={1,2,3,4}X = \{1,2,3,4\}X={1,2,3,4} and Y={2,4,7,9}Y=\{2,4,7,9\}Y={2,4,7,9}. Their union is
X∪Y={1,2,3,4,7,9} X \cup Y = \{1,2,3,4,7,9\}X∪Y={1,2,3,4,7,9} and their intersection is X∩Y={2,4}.X \cap Y = \{2,4\}.X∩Y={2,4}. Notice that we do not repeat elements in a set. An element is either in the set or it isn’t. (A generalization, the multiset, will be developed in Chapter 11.) Therefore, ∣X∪Y∣|X \cup Y|∣X∪Y∣ (recall, the number of elements in the union) won’t just be ∣X∣+∣Y∣|X|+|Y|∣X∣+∣Y∣.

Theorem 1.3.3. If XXX and YYY are finite sets, then ∣X∪Y∣=∣X∣+∣Y∣−∣X∩Y∣.|X \cup Y| = |X|+|Y|-|X\cap Y|.∣X∪Y∣=∣X∣+∣Y∣−∣X∩Y∣.

Proof. If an element is only in XXX, it will be counted by ∣X∣|X|∣X∣. If an element is only in YYY, it will be counted by ∣Y∣|Y|∣Y∣. Therefore any elements in X∩YX \cap YX∩Y will be counted twice. There are exactly ∣X∩Y∣|X \cap Y|∣X∩Y∣ of these elements, so subtracting that amount gives ∣X∪Y∣|X \cup Y|∣X∪Y∣. □\square□

Definition 1.3.4. Let XXX and YYY be sets. The difference X−YX-YX−Y is the set of all elements in XXX but not YYY; i.e. X−Y={ x ∣ x∈X and x∉Y}.X-Y=\{ \, x \, | \, x \in X \text{ and } x \not\in Y\}.X−Y={x∣x∈X and x​∈Y}.

Example. Like before, let X={1,2,3,4}X = \{1,2,3,4\}X={1,2,3,4} and Y={2,4,7,9}Y=\{2,4,7,9\}Y={2,4,7,9}. Then X−Y={1,3}X-Y=\{1,3\}X−Y={1,3}. Notice that Y−X={7,9}Y-X=\{7,9\}Y−X={7,9}; just like with numbers, taking a diference is not commutative. (That is, the order matters.)

Often when working with sets there is a universal or ambient set, sometimes denoted Ω\OmegaΩ, that all of our sets are assumed to be subsets of. This set is typically inferred from context.

  • In calculus, this set might be R\mathbf{R}R.
  • If my set is “students taking MATH 174,” my universal set might be the set of all students at Coastal Carolina.
  • In probability theory, the sets are events (e.g. “the roll is odd”) and Ω\OmegaΩ is the sample space (e.g. all possible rolls of the die).

Definition 1.3.5. Let XXX be a set contained in some universal set Ω\OmegaΩ. The complement of XXX (relative to Ω\OmegaΩ), denoted X‾\overline{X}X, is all the elements not in XXX. In other words, X‾=Ω−X\overline{X}=\Omega-XX=Ω−X.

Example. Suppose X={a,b,c}X=\{a,b,c\}X={a,b,c} is contained in the universal set Ω={a,b,c,d,e}\Omega=\{a,b,c,d,e\}Ω={a,b,c,d,e}. Then X‾={d,e}\overline{X}=\{d,e\}X={d,e}.

Definition 1.3.6. Let XXX and YYY be sets. Their Cartesian product (in this book, just “product”) is the set X×Y={(x,y) ∣ x∈X and y∈Y}.X\times Y = \{(x,y)\,|\, x \in X \text{ and } y \in Y\}.X×Y={(x,y)∣x∈X and y∈Y}. The elements of the product are called ordered pairs.

As the name implies, the ordering of the entries in an ordered pair matters. Therefore, we will adopt the convention that unordered structures are enclosed in {\{{ curly braces }\}} and ordered structures are enclosed in ((( parentheses ))). The immediate consequence to this observation is that X×YX \times YX×Y is different from Y×XY \times XY×X.

1.4 Operations involving more than two sets
